## 📌 Overview

This project explores different approaches to handwritten digit classification using the MNIST dataset.
It compares traditional machine learning with deep learning models to understand performance differences.

---

## 🎯 Objectives

* Build and compare multiple classification models
* Analyze performance across different approaches
* Deploy a working digit classifier

---

## ⚙️ Models Implemented

### 📉 Logistic Regression (Deployed)

* Lightweight and fast
* Used for live deployment
* Works efficiently in constrained environments

---

### 🧠 Artificial Neural Network (ANN)

* Learns non-linear relationships
* Improves accuracy over baseline

---

### 🔥 Convolutional Neural Network (CNN)

* Best suited for image data
* Achieves highest accuracy (~99%)
* Captures spatial features effectively

---

## 🏆 Results

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | ~96%  |
| ANN                 | ~98%     |
| CNN                 | ~99%     |

---

## 🧠 Key Insights

* Simple models struggle with complex patterns
* Neural networks improve performance significantly
* CNN performs best due to spatial feature extraction
